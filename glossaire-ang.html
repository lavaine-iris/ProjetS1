Glossary

Abaya : An abaya is a veil that covers the whole body except the face, hands and feet. It can be complemented by a niqab which is a full veil that covers only the face except the eyes.

Burqa : the burqa is a full veil that covers the whole body, even the eyes, so you can see that there is a grid at eye level.

Kabul : Kabul is the capital of Afghanistan.

Taliban : The Taliban are members of an Islamic movement.

NGO (Non-Governmental Organisation) : A non-governmental organisation is an organisation that provides humanitarian aid.

Afghanistan : Afghanistan is a country in Central Asia.
